Welcome to the Terraria NPC Housing Tracker!

This is a tool to help you keep track of your NPCs and their housing in Terraria.
It allows you to easily see which NPCs are currently housed and which ones are homeless.

(The point of this is to help you keep track of which NPCs
you've built specific houses for, and which ones you haven't.)

Most Terraria mods that add NPCs are also supported.
I hope you find it useful!

How to Use

To use the tracker, simply enable whichever mods you have installed and click the "Load NPCs" button.
This will show all relevant NPCs in the table below.
The checkboxes next to each NPC allow you to mark them as housed or homeless.
